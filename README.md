# Chara Beauty Website

## Deskripsi Project
Chara Beauty merupakan website sederhana bertema toko kecantikan (beauty store) yang menampilkan berbagai produk skincare dan makeup.
Website ini dibuat menggunakan HTML, CSS, dan JavaScript dengan tampilan yang sederhana namun interaktif seperti e-commerce.

## Fitur Website
Beberapa fitur yang tersedia pada website ini:

- Menampilkan daftar produk kecantikan
- Fitur pencarian produk
- Keranjang belanja (shopping cart)
- Menambahkan produk ke keranjang
- Menghapus produk dari keranjang
- Menampilkan total harga produk
- Tampilan produk menggunakan grid layout
- Navigasi menu (Home, Tentang Kami, Produk, Kontak)

## Tema Project
Tema yang digunakan dalam project ini adalah **Beauty / Cosmetic Store Website** yang menampilkan produk skincare dan makeup dengan konsep tampilan sederhana seperti toko online.
